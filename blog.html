What is the difference between map(), filter(), find() and forEach() methods?
  • The map() method is used to iterate each element of an array and return a new array with the same length as the original array.
  • The filter() method is used to logically iterate each element of an array in which's elements pass a given logic.
  • The find() method is used to logically iterate each element of an array and it's stored only one element which element pass a given logic and then stops iterate execution.
  • The forEach() method is used to loop over the elements of an array and perform an action for each element in an array and does not return anything.
What is the difference between Local Storage, Session Storage and Cookies?
  • The localStorage is used to store data in the user's browser and the data stored in local storage does not expire.
  • The sessionStorage is used to store data in the user's browser but the data is only available during the current browser session. When the user closes their browser the session storage data is cleared.
  • The cookies help websites remember users and their preferences, which can improve their browsing experience and cookies can also be used to track user behavior and gather personal information, which has raised privacy concerns.
What is an event loop and how does its work?

The event loop allows JavaScript to handle multiple tasks simultaneously without blocking the main thread, which is crucial for building responsive and efficient web applications.
In simple terms, the event loop continuously checks for tasks that are ready to be executed and adds them to the call stack. If the call stack is empty, the event loop waits for new tasks to be added. When a task is added to the call stack, it is executed, and any other tasks that are waiting in the queue can be executed in the same manner.

What does quizTimer() do?

This function calculates the quiz execution time and notifies it 60 seconds after the quiz execution time starts and blinking every second after 60 seconds.

What does chooseQuiz() do?

This function picks the selected answer for each quiz and calls a function to display the position of the answer.
